- Email 1 CTA: Test "paixao.ca/founding" link vs a button "Become a Founding Member"

**Deliverability checklist:**
- SPF record set for paixao.ca ✓
- DKIM signature configured ✓
- DMARC policy set (p=quarantine minimum) ✓
- Unsubscribe link in every email ✓
- Physical mailing address in footer (required by CASL) ✓
- Confirmed opt-in on waitlist signup ✓

**CASL compliance note (Canada):**
Canadian Anti-Spam Legislation requires: express consent (your signup form provides this), sender identification, unsubscribe mechanism, physical mailing address. Ensure all five emails include a footer with these elements.
